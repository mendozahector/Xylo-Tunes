# Xylo Tunes
Using Apple's APIs to tap into audio capabilities.

- Users are able to play with a Xylophone
- Users are able to record tunes (saved with Realm)

### Skills
* AVFoundation
* Auto Layout
* CocoaPods
* Persistent Local Data Storage (Realm)
